# The Unknowable

A modern, elegant website for the operatic ballet "The Unknowable: An Operatic Ballet in Two Acts" composed by Benjamin T. Rossen.

## About the Production

**The Unknowable** is a groundbreaking operatic ballet that premiered at Sanders Theatre in February 2024 as Lowell House Opera's 78th production. The work explores the profound complexity of making difficult life decisions through an innovative fusion of opera and ballet.

### Creative Team

- **Composer & Librettist**: Benjamin T. Rossen '23
- **Production Company**: Lowell House Opera, Harvard University
- **Premiere**: Sanders Theatre, Cambridge, MA - February 10-11, 2024

### Musical Foundation

The work interweaves original compositions with two masterpieces of the song cycle repertoire:
- **Les nuits d'été** by Hector Berlioz
- **Lieder eines fahrenden Gesellen** by Gustav Mahler

## Website Features

This website showcases the production with:

- **Modern, Responsive Design**: Fully responsive layout that works on all devices
- **Smooth Animations**: Elegant scroll animations and interactive hover effects
- **Rich Content Sections**:
  - Hero section with production details
  - About the work and its innovative approach
  - Musical foundation and influences
  - Creative team information
  - Press reviews and media coverage
- **Performance Optimized**: Fast loading with efficient JavaScript
- **Accessibility**: Keyboard navigation support and semantic HTML

## Technical Stack

- **HTML5**: Semantic markup for better SEO and accessibility
- **CSS3**: Modern styling with:
  - CSS Grid and Flexbox for layouts
  - CSS Custom Properties (variables) for theming
  - Smooth transitions and animations
  - Responsive design with media queries
- **Vanilla JavaScript**: No dependencies, pure JS for:
  - Smooth scrolling
  - Scroll animations
  - Interactive card effects
  - Parallax effects
  - Active navigation highlighting

## Running the Website

Simply open `index.html` in a modern web browser. No build process or server required.

For local development with live reloading, you can use:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx serve

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000` in your browser.
